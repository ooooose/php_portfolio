## ■サービス概要
WorksHub

## ■概要
個人が作ったポートフォリオを登録するアプリ。

## ■実装予定の機能
 - ユーザー登録・削除機能
 - ログイン・ログアウト機能
 - 作品のCRUD機能
 - 作品の検索機能
 - お気に入り機能
