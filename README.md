## ■サービス概要
BORAD APP

## ■概要
基礎編のようなBOARD APPをphp/Laravelで作成する。

## ■実装予定の機能
 - ユーザー登録・削除機能
 - ログイン・ログアウト機能
 - BOARDのCRUD機能
 - TODOの検索機能
 - コメント機能、お気に入り機能　など
